# 毕业设计-体检录入系统

#### 介绍
本体检信息录入系统采用前端：vue；后端：springboot+mybatis-plus+redis+mysql技术架构开发，前后端分离，容易上手。除了基本的体检结果查询、录入及导出外，在录入中还能对录入信息进行智能计算。
